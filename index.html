⌨️ User Guide

  • Voice Input: Starts transcription at a new line at the end of the file.
  • Save (Ctrl+S): Opens the "Save As" window and tracks in browser history.
  • Timestamps: When ON, press Enter to tag the current recording time.
  • Focus: The cursor automatically returns to your last position after clicking any tool.
  • Language 1: Select the language for Speech-to-Text transcription.
  • Convert: Translates your entire editor content into the Second Language option.
  • Speak: Reads the text aloud. If you convert the text, the voice output will automatically switch to that language.
  • Note: If using a new language for the first time, voice output may take a moment to initialize.
