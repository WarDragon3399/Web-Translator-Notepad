# Code of Conduct

## Our Goal
The Web Translator Notepad is a personal project built for helpfulness and efficiency. We want to maintain a community that is respectful, professional, and kind.

## Expected Behavior
* **Be Respectful:** Treat others as you would like to be treated.
* **Be Constructive:** If you find a bug or have a suggestion, please share it politely.
* **Be Kind:** Harassment, trolling, or insulting language will not be tolerated.

## Scope
This applies to all interactions within this repository, including Issues, Pull Requests, and Discussions.

## Contact
If you encounter any behavior that violates these simple rules, please reach out to the project owner at: **parthrathodtk@gmail.com**.
